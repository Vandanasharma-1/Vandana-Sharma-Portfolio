Credit Risk Prediction System (AUC: 0.93)

Developed a robust end-to-end machine learning system for predicting credit risk and supporting financial institutions in data-driven loan approval decisions. Multivariate Time-Series Forecasting using LSTM

Developed a deep learning based forecasting system to model and predict complex multivariate financial time-series indicators using Long Short-Term Memory (LSTM) neural networks. The LSTM architecture was optimized using dropout regularization and the Adam optimizer to improve generalization and reduce overfitting. Model performance was analyzed across different forecasting horizons to evaluate prediction stability and identify optimal time intervals for forecasting. This project demonstrates the effectiveness of deep learning techniques for modeling complex sequential financial data.
